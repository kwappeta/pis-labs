# Use Case: Добавление отзыва и завершение чтения книги

**Вариант:** №19 
**Актор:** Авторизованный пользователь

## 1. Краткое описание
Пользователь переводит книгу из текущего списка чтения в архив прочитанного, одновременно выставляя оценку и оставляя текстовый отзыв.

## 2. Основной поток событий (Happy Path)
1. Пользователь переходит в раздел «Мои книги» и выбирает вкладку «Читаю».
2. Пользователь выбирает конкретную книгу и нажимает кнопку «Завершить чтение».
3. Система отображает форму с полями:
    - Рейтинг (звезды от 1 до 5);
    - Текстовое поле для отзыва (необязательно).
4. Пользователь выставляет рейтинг, пишет отзыв и нажимает «Сохранить».
5. **Система выполняет транзакцию:**
    - Сохраняет отзыв в базе данных;
    - Меняет статус книги на «Прочитано»;
    - Обновляет дату завершения чтения;
    - Пересчитывает средний рейтинг книги для всех пользователей.
6. Система закрывает форму и выводит уведомление об успешном обновлении.

## 3. Альтернативные потоки
### А1: Отмена действия
1. На шаге 3 или 4 пользователь нажимает кнопку «Отмена».
2. Система закрывает форму без сохранения изменений. Статус книги остается «Читаю».

## 4. Исключительные ситуации (Error Path)
### Е1: Ошибка валидации рейтинга
1. На шаге 4 пользователь пытается нажать «Сохранить», не выбрав количество звезд.
2. Система блокирует отправку и подсвечивает поле рейтинга красным цветом.
